## WEBOIN INTERNSHIP ASSESSMENT

I am Saif Ali Khan. I am developing this single page website as an assessment for the company Weboin. 
I have used Next.JS framework to make this possible.
I have utilized Typescript for coding and Tailwind CSS for styling. I have also used 3D webGL shader 
for the background. It is fully responsive and can be run on any device.
I have used Lenis Scroll for smooth scrolling. 

## Setup

1. **Clone the repository and navigate to the project directory:**
   ```bash
   git clone <repository_url>
   cd weboin
   ```

2. **Install project dependencies:**
   ```bash
   npm install
   ```

3. **Launch the local development server:**
   ```bash
   npm run dev
   ```
